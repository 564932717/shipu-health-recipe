# 接口说明

后端基础地址：

```text
http://localhost:8080
```

统一返回结构：

```json
{
  "success": true,
  "message": "ok",
  "data": {}
}
```

## 演示账号

```text
username: demo
password: 123456
userId: demo
```

## 用户接口

### 注册

`POST /api/users/register`

```json
{
  "username": "demo2",
  "password": "123456",
  "displayName": "演示用户2"
}
```

### 登录

`POST /api/users/login`

```json
{
  "username": "demo",
  "password": "123456"
}
```

## 健康档案接口

### 查询健康档案

`GET /api/profiles/demo`

### 保存健康档案

`POST /api/profiles`

```json
{
  "userId": "demo",
  "age": 22,
  "gender": "MALE",
  "heightCm": 175,
  "weightKg": 72,
  "goal": "FAT_LOSS",
  "chronicConditions": ["轻度高血压"],
  "taboos": ["花生", "高盐"],
  "tastePreferences": ["清淡", "高蛋白"]
}
```

## 健康评估接口

### 计算 BMI、热量与营养目标

`POST /api/health/evaluate`

请求体同健康档案。

## 三餐推荐接口

### 生成一日三餐

`POST /api/health/recommendations/day`

请求体同健康档案。返回早餐、午餐、晚餐、加餐、预估摄入和 AI 建议。

## 食谱接口

### 查询食谱

`GET /api/recipes?keyword=鸡胸&goal=FAT_LOSS&mealType=LUNCH`

参数均可选。

### 食谱详情

`GET /api/recipes/B001`

## 饮食记录接口

### 新增饮食记录

`POST /api/diet-records`

```json
{
  "userId": "demo",
  "foodName": "糙米鸡胸蔬菜碗",
  "mealType": "LUNCH",
  "calories": 560,
  "proteinGram": 42,
  "fatGram": 14,
  "carbohydrateGram": 66
}
```

### 查询用户饮食记录

`GET /api/diet-records?userId=demo`

### 今日营养统计

`GET /api/diet-records/summary?userId=demo&targetCalories=1875`

## 健康周报接口

### 查询本周报告

`GET /api/reports/week?userId=demo&targetCalories=1875`
