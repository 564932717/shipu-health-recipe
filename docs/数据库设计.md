# 数据库设计

当前代码第一版使用内存模拟数据，正式版本可按以下表结构迁移到 MySQL。

## 用户表 user_account

| 字段 | 类型 | 说明 |
|---|---|---|
| id | varchar(64) | 用户 ID |
| username | varchar(64) | 登录名 |
| password | varchar(128) | 密码，正式版应加密 |
| display_name | varchar(64) | 昵称 |
| created_at | datetime | 创建时间 |

## 健康档案表 health_profile

| 字段 | 类型 | 说明 |
|---|---|---|
| id | bigint | 主键 |
| user_id | varchar(64) | 用户 ID |
| age | int | 年龄 |
| gender | varchar(20) | 性别 |
| height_cm | decimal(6,2) | 身高 |
| weight_kg | decimal(6,2) | 体重 |
| goal | varchar(40) | 健康目标 |
| chronic_conditions | varchar(255) | 慢病情况，逗号分隔 |
| taboos | varchar(255) | 忌口，逗号分隔 |
| taste_preferences | varchar(255) | 口味偏好，逗号分隔 |
| updated_at | datetime | 更新时间 |

## 食谱表 recipe

| 字段 | 类型 | 说明 |
|---|---|---|
| id | varchar(64) | 食谱编号 |
| name | varchar(128) | 食谱名称 |
| meal_type | varchar(20) | 早餐、午餐、晚餐、加餐 |
| category | varchar(64) | 分类 |
| suitable_goals | varchar(255) | 适用目标 |
| tags | varchar(255) | 标签 |
| ingredients | text | 食材 |
| steps | text | 制作步骤 |
| calories | int | 热量 |
| protein_gram | int | 蛋白质 |
| fat_gram | int | 脂肪 |
| carbohydrate_gram | int | 碳水 |

## 饮食记录表 diet_record

| 字段 | 类型 | 说明 |
|---|---|---|
| id | varchar(64) | 记录 ID |
| user_id | varchar(64) | 用户 ID |
| food_name | varchar(128) | 食物名称 |
| meal_type | varchar(20) | 餐别 |
| calories | int | 热量 |
| protein_gram | int | 蛋白质 |
| fat_gram | int | 脂肪 |
| carbohydrate_gram | int | 碳水 |
| eaten_at | datetime | 记录时间 |

## 收藏表 recipe_favorite

| 字段 | 类型 | 说明 |
|---|---|---|
| id | bigint | 主键 |
| user_id | varchar(64) | 用户 ID |
| recipe_id | varchar(64) | 食谱编号 |
| created_at | datetime | 收藏时间 |
